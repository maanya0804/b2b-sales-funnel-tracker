# B2B Sales Funnel Conversion Tracker

Modelled a 5-stage consultative sales funnel across IT, BFSI, and Manufacturing 
verticals to identify conversion leakage points and surface actionable recommendations.

## What this does
- Simulates deal flow across Prospect → Engage → Propose → Close → Retain
- Calculates stage-by-stage conversion rates for 3 industry verticals
- Identifies the weakest conversion stage per vertical
- Generates cross-vertical recommendations for pipeline improvement

## Tools
Python, Pandas, Matplotlib

## Output
- chart1_funnel.png — deal flow funnel per vertical
- chart2_conversion_rates.png — stage-by-stage conversion comparison
- chart3_overall_conversion.png — end-to-end conversion by vertical
- sales_funnel_brief.txt — analyst recommendations brief
